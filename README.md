# Multiplication Table Generator

A simple web-based tool to generate multiplication tables dynamically using JavaScript.

## Features
- **Dynamic Calculation:** Generates tables from 1 to 10 for any input number.
- **Clean UI:** Displays results neatly inside the result container without page refreshes.
- **Validation:** Ensures users enter valid numbers before calculation.

## How to use
1. Open `https://muhammadzaid-kh.github.io/JS-Multiplication-Table-Generator/` in your browser.
2. Enter a number in the input box.
3. Click "Check" to see the table.

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
